Desigualdad de Minkowsky

Sea \(p\) número real tal que \(1 \leq p < \infty \; \). Dados las sucesiones o funciones \(x, \;y \; ; \) se tiene (dependiendo del caso y siempre y cuando las series e integrales ubicadas a la derecha de cada desigualdad converjan):

\[ \left ( \sum_{i=1}^{n} \left | x_i \pm y_i \right |^p \right )^{1/p} \leq \left ( \sum_{i=1}^{n} \left |x_i \right |^p \right )^{1/p} + \left ( \sum_{i=1}^{n} \left |y_i \right |^p \right )^{1/p} \]

\[ \left ( \sum_{i=1}^{\infty} \left | x_i \pm y_i \right |^p \right )^{1/p} \leq \left ( \sum_{i=1}^{\infty} \left |x_i \right |^p \right )^{1/p} + \left ( \sum_{i=1}^{\infty} \left |y_i \right |^p \right )^{1/p} \]

\[ \left ( \int_{\Omega} \left | x_i \pm y_i \right |^{p} \right )^{1/p} \leq \left ( \int_{\Omega} \left |x_i \right |^p \right )^{1/p} + \left ( \int_{\Omega} \left |y_i \right |^p \right )^{1/p} \]

Usando la siguiente notación de norma (dependiendo del caso):

\[ \|x\|_p = \left ( \sum_{i=1}^{n} \left |x_i \right|^p \right )^{1/p},\;\; \|x \|_p = \left ( \sum_{i=1}^{\infty} \left |x_i \right|^p \right )^{1/p}, \;\; \|x \|_p = \left ( \int_{\Omega} \left |x_i \right|^p \right )^{1/p} \]

Las desigualdades se resumen como:

\[ \|x \pm y \|_p \leq \| x\|_p + \|y \|_p \]

Dem.

El caso \(p = 1\) es directo. Para \(p>1\) usaremos la desigualdad de Holder para la demostración. Consideremos primero la siguiente factorización (para \(a, \; b \in \mathbb{R}\)).

\[ (a + b)^p = a(a+b)^{p - 1} + b(a+b)^{p - 1} \]

Considerando \(a = |x_i| , \; b = |y_i|\) y aplicando suma tendremos: \[ \sum_{i=1}^{n} \left ( |x_i| + |y_i| \right )^p = \sum_{i=1}^{n} |x_i|\left ( |x_i|+|y_i|\right )^{p - 1} + \sum_{i=1}^{n} |y_i|\left ( |x_i|+|y_i|\right )^{p - 1} \]

Aplicando desigualdad de Hölder a ambas sumas de la derecha (considerando \(q\) tal que \(p^{-1} + q^{-1} = 1 \) ):

\[ \sum_{i=1}^{n} \left ( |x_i| + |y_i| \right )^p \leq \|x\|_p \left ( \sum_{i=1}^{n} \left ( |x_i|+|y_i|\right )^{(p - 1)q} \right )^{1/q} + \|y\|_p \left ( \sum_{i=1}^{n} \left ( |x_i|+|y_i|\right )^{(p - 1)q} \right )^{1/q} \]

Notemos que \( (p-1)q = p\). Dividiendo por \( \left ( \sum_{i=1}^{n} \left ( |x_i|+|y_i|\right )^{p} \right )^{1/q} \) a ambos lados de la desigualdad:

\[ \left ( \sum_{i=1}^{n} \left ( |x_i| + |y_i| \right )^p \right )^{1/p} \leq \|x\|_p + \|y\|_p \]

\[ \therefore \|x \pm y \|_p \leq \|x\|_p + \|y\|_p \]

Nota: si \( \left ( \sum_{i=1}^{n} \left ( |x_i|+|y_i|\right )^{p} \right )^{1/q} = 0 \) se obtiene la desigualdad de forma trivial. \(\square\)
